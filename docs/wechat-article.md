# 我做了一个 Chrome 扩展，让你像编辑 Word 一样编辑任何网页

我经常遇到一个问题：看到一个网页，想改几个字保存下来，或者想调整一下排版再导出，却发现没有一个顺手的工具。

打开开发者工具手动改 DOM？太麻烦，而且改完刷新就没了。截图用 PS 改？文字一多就崩溃。Ctrl+S 保存网页？格式乱得没法看。

我想要的很简单：**直接在网页上点哪改哪，改完能保存。**

找了一圈没找到满意的方案，于是自己动手写了一个 Chrome 扩展 —— **HTML Visual Editor**。

## 它解决什么问题？

一句话：**让你在浏览器里像用 Word 一样编辑网页，编辑完可以导出保存。**

不用打开开发者工具，不用懂 HTML，不用装任何额外软件。装上扩展，按一下 Alt+E，整个网页就变成了你的编辑器。

## 具体能做什么？

### 点击选中，双击编辑

进入编辑模式后，单击任何元素会出现蓝色选中框，双击文字就能直接修改内容。改标题、改段落、改按钮上的文字，都是双击就改，跟编辑文档一模一样。

### 可视化调样式

选中元素后会弹出样式面板，颜色、字号、间距、边框这些属性全都可以直接调。不用记任何 CSS 属性名，点点鼠标就能把页面改成你想要的样子。

### 拖拽重新排版

段落顺序不满意？直接拖。图片位置不对？也是拖。所有元素都支持拖拽重新排列。

### 表格也能编辑

遇到网页上的表格，可以直接增删行列、编辑单元格内容，操作方式和 Excel 差不多。

### 给网页加批注

这个功能是我自己最常用的。选中一段文字可以添加批注评论，也可以在页面上贴便签。做内容审查、标记修改意见、阅读笔记，都很方便。

### 编辑完一键导出

这是整个工具的核心价值 —— 编辑完的网页可以直接导出为 HTML 文件，排版样式全部保留。也可以一键复制 HTML 代码。批注内容支持单独导出为 JSON，方便下次导入继续使用。

## 哪些场景用得上？

说几个我自己和朋友实际在用的场景：

**保存网页内容** —— 看到好文章想收藏，先编辑掉广告和无关内容，再导出保存，比直接 Ctrl+S 干净太多

**竞品分析截图** —— 改几个字、换个颜色就能出对比图，不用开 PS

**教学备课** —— 老师把网页教材重新编排、加上批注，导出后直接打印

**前端原型调整** —— 不用改代码重新构建，直接在页面上调效果看看

## 怎么安装？

目前还没有上架 Chrome 应用商店，需要从源码构建安装：

1. 到 GitHub 下载源码
2. 运行 `npm install` 和 `npm run build`
3. 打开 Chrome 的 `chrome://extensions/`，开启开发者模式
4. 点「加载已解压的扩展程序」，选择 dist 目录

安装完成后，打开任意网页，按 **Alt+E** 就能开始编辑。
